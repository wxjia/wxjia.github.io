Remove Bad Vim from Newly Installed Ubuntu System

If you installed a new ubuntu system, you must bother with the bad vim.

Now, let me teach you how to remove the bad one and install a better one.
Read after me: So Easy!
Just execute these two command:

sudo apt-get remove vim-common
sudo apt-get install vim

Open the vim editer, well done.
